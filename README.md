# Zoom_Auto_Join
Automatically checks meeting code and time to join from excel and joins the zoom meet without any human intervention.


# Branch Special: 
  Has a learner bot that will learn coordinates of action for your computer and will let the main program know. 
  
 
 # Guide to use:
 1. Download repo.
 2. Run `learnerbot.py` and follow it's intructions. (Note: Provide full zoom application path for better working)
 3. Run `main.py` from next time.
 
 
 # Prereqs: 
 Must have Python installed.
